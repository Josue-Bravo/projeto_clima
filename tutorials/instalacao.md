# Instalação e Uso

### Pré-requisitos
- Navegador moderno (Chrome, Firefox, Edge, Safari)
- Node.js 18+ (apenas para rodar os testes)

### Instalação
1. Clone o repositório
2. Abra o `index.html` no navegador — não precisa de servidor

### Rodando os testes
```bash
npm install
npm test
```
### Gerando a documentação
```bash
npm run docs
```
