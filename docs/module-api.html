Module: api

Módulo de comunicação com as APIs Open-Meteo. Parte da aplicação de Previsão do Tempo — um site que permite ao usuário consultar o clima atual e a previsão dos próximos 7 dias de qualquer cidade do mundo, sem necessidade de autenticação.

Responsável por validação de entrada, busca de coordenadas geográficas e recuperação de dados meteorológicos.

(async, static) fetchCoordinates(city) → {Promise.<{name: string, latitude: number, longitude: number}>}

Busca as coordenadas geográficas de uma cidade pelo nome utilizando a API de Geocoding do Open-Meteo.

Parameters:
Name Type Description
city string

Nome da cidade já validado (sem espaços extras).
Source:

Throws:

  • Se a requisição HTTP falhar (status não-ok).

    Type
    Error

  • Se nenhum resultado for encontrado para o nome informado.

    Type
    Error
Returns:

Objeto com o nome oficial, latitude e longitude da cidade encontrada.

Type
Promise.<{name: string, latitude: number, longitude: number}>
Example
const coords = await fetchCoordinates('Manaus');
console.log(coords.latitude);  // -3.10194
console.log(coords.longitude); // -60.025
console.log(coords.name);      // 'Manaus'

await fetchCoordinates('xyzabcqwerty');
// lança Error: 'Cidade não encontrada. Tente novamente.'

(async, static) fetchWeather(latitude, longitude) → {Promise.<{current_weather: {temperature: number, windspeed: number, weathercode: number}, hourly: {relativehumidity_2m: Array.<number>}, daily: {time: Array.<string>, weathercode: Array.<number>, temperature_2m_max: Array.<number>, temperature_2m_min: Array.<number>}}>}

Busca os dados meteorológicos completos de uma localização geográfica utilizando a API de previsão do Open-Meteo.

Retorna clima atual (temperatura, vento, código do clima), umidade horária e previsão diária para os próximos 7 dias.

Latitude da localização.

    Type

    Type

Objeto com dados meteorológicos completos.

Type
Promise.<{current_weather: {temperature: number, windspeed: number, weathercode: number}, hourly: {relativehumidity_2m: Array.<number>}, daily: {time: Array.<string>, weathercode: Array.<number>, temperature_2m_max: Array.<number>, temperature_2m_min: Array.<number>}}>
